Filecoin Slingshot

Our project:

  1. Download Google Open Images machine learning dataset.
  2. Restructure the loosely coupled datasets into industry-standard archival packaging based on schema.org linked data in JSON-LD format and Bagit containers.
  3. Upload to Filecoin Testnet as part of Slingshot competition.
  4. Build a map overlay app from extracted GPS data for the Slingshot competition. Explore relationships between machine learning annotations and real-world settings.
  5. Train Deplatformr models on personal datasets using Open Image dataset.
